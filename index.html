Here We Go!

Group Project

Here We Go! is a single-page, travel resource web application. The user inputs a destination through the autocomplete dropdown, and the app provides information including tourist attraction, location, weather, visual, currency, and time zone data. The focus of this project was utilizing external APIs and dynamically generating the front end with jQuery. The project also features a robust user interface through HTML, CSS, and bootstrap.

Energy Logan Application

Group Project

Energy Logan Application (ELA) is an energy services company workflow tool for companies that deal in surveying buildings to determine which lighting fixtures can be replaced with more energy efficient fixtures, earning them federal incentives and energy cost savings. The purpose of the application is to streamline the connection between the office and the surveying technician. The application allows the office to create new projects which on site technicians can immediately start adding survey. The focus of this project was building a backend through MySQL, Express, Node.js, Sequelize, and Handlebars - providing an intuitive and easy to use user interface to communicate with the backend database.

Backstage Pass Trivia

Homework

Backstage Pass Trivia is an in-browser game featuring trivia questions about the members of Bakusute Sotokanda Icchome. The frontend runs off of bootstrap, css, and css animations. The script involves storing the trivia questions and answers, grabbing user input, and comparing the two with win/loss logic. The script also utilizes timers, intervals, and jQuery to render the results page.
